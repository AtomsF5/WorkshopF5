## ¡Rompiendo Código!



- Install repository:
```
git clone https://github.com/AtomsF5/WorkshopF5.git
```

```
Create archive to the project  .env
```

- Install composer:
```
composer install
```

```
php artisan key:generate
```

- Install nodejs:
```
npm install
```

- Run npm:
```
npm run dev
```

- Run project:
```
$ php artisan serve
```
